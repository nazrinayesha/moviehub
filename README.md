# Movie Search App

A React-based movie search application that allows users to search for
movies and view movie information using the TMDB API.

## Features

-   Search movies by title
-   Display movie posters and details
-   Responsive movie card layout
-   Clean and simple user interface
-   API integration with TMDB

## Tech Stack

-   React.js
-   JavaScript(ES6+)
-   Axios
-   HTML5
-   CSS3
-   TMDB API

## Installation

Clone the repository:

``` bash
git clone https://github.com/nazrinayesha/movie_explorer_application.git
```

Go to the project folder:

``` bash
cd movie_explorer_application
```

Install dependencies:

``` bash
npm install
```

Run the application:

``` bash
npm start
```

The app will run on:

    http://localhost:5173
